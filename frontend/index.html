What happens when you click Run Inference:
1. Browser POSTs to /api/predict/transaction/{index}
2. FastAPI fetches the row from test_features.parquet
3. NaN → -999, cast to FP32
4. FastAPI calls triton:8000/v2/models/fraud_detection_xgboost/infer
5. Triton's Python backend runs xgb.predict_proba()
6. Fraud probability flows back to the browser

Batch Inference Performance

Send N transactions to Triton in a single request to measure GPU batch throughput. Triton's dynamic batching merges concurrent single requests automatically — this test exercises explicit large-batch inference.
